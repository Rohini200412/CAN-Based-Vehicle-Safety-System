# CAN-Based-Vehicle-Safety-System
CAN-based automotive safety and monitoring system using LPC2129

## Description
This project implements an automotive safety and monitoring system using CAN protocol and LPC2129 microcontroller.

## Features
- Engine temperature monitoring using DS18B20
- Left and right indicator control
- Reverse obstacle detection using ultrasonic sensor
- CAN communication between multiple nodes
- Fault tolerant system design

## Hardware Used
- LPC2129
- MCP2551 CAN Transceiver
- DS18B20
- HC-SR04 Ultrasonic Sensor
- LCD
- LEDs
- Switches

## Software Used
- Embedded C
- Keil uVision
- Flash Magic
